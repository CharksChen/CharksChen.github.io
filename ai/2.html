一、什么是机器学习

1.1 定义与核心思想

机器学习（Machine Learning）是人工智能的核心分支。Arthur Samuel（1959）将其定义为："赋予计算机无需显式编程即可学习的能力的研究领域。" Tom Mitchell（1997）给出了更形式化的定义：对于某类任务 $T$ 和性能度量 $P$，一个计算机程序被认为从经验 $E$ 中学习，当它在任务 $T$ 中的性能（由 $P$ 衡量）随着经验 $E$ 的增加而提高。

传统编程与机器学习的本质区别：

  • 传统编程：规则 + 数据 → 答案
  • 机器学习：数据 + 答案 → 规则

1.2 三大学习范式

范式数据需求核心任务典型算法
监督学习带标签数据 $(x_i, y_i)$分类、回归线性回归、逻辑回归、SVM、决策树、神经网络
无监督学习无标签数据 $x_i$聚类、降维、密度估计K-Means、DBSCAN、PCA、t-SNE
强化学习交互经验 $(s, a, r, s')$序贯决策Q-Learning、Policy Gradient、PPO

1.3 发展简史

  1. 推理期（1950s-1970s）：符号主义AI，逻辑推理为主。1950年图灵提出"图灵测试"，1956年达特茅斯会议标志AI诞生。
  2. 知识期（1970s-1980s）：专家系统兴起，将人类专家知识编码为规则。
  3. 统计学习期（1990s-2010s）：SVM、决策树、集成学习蓬勃发展，数学理论日趋成熟。
  4. 深度学习期（2012-至今）：AlexNet（2012）引爆深度神经网络浪潮，GPT系列开启大模型时代。
关键洞察：机器学习不是魔法，而是一套基于数据、通过优化目标函数来学习模式的数学方法。理解这一点比记住任何算法都重要。

三、数学基础（上）：线性代数

3.1 标量、向量与矩阵

机器学习中，数据通常以矩阵形式表示。一个包含 $m$ 个样本、$n$ 个特征的数据集可表示为矩阵 $\mathbf{X} \in \mathbb{R}^{m \times n}$。

  • 标量：单个数值，如 $a \in \mathbb{R}$
  • 向量：一维数组，如 $\mathbf{x} = [x_1, x_2, \ldots, x_n]^T$
  • 矩阵：二维数组，如 $\mathbf{X} = \begin{bmatrix} x_{11} & x_{12} \\ x_{21} & x_{22} \end{bmatrix}$

3.2 矩阵运算

矩阵乘法：若 $\mathbf{A} \in \mathbb{R}^{m \times n}$，$\mathbf{B} \in \mathbb{R}^{n \times p}$，则 $\mathbf{C} = \mathbf{A}\mathbf{B} \in \mathbb{R}^{m \times p}$，其中：

$$c_{ij} = \sum_{k=1}^{n} a_{ik} b_{kj}$$

3.3 范数

范数用于衡量向量的大小，在正则化中起到关键作用。

  • $L_1$ 范数：$\|\mathbf{x}\|_1 = \sum_i |x_i|$，产生稀疏解，用于特征选择
  • $L_2$ 范数：$\|\mathbf{x}\|_2 = \sqrt{\sum_i x_i^2}$，防止过拟合，在岭回归和权重衰减中使用

3.4 特征值与特征向量

对于方阵 $\mathbf{A}$，若 $\mathbf{A}\mathbf{v} = \lambda \mathbf{v}$，则 $\lambda$ 为特征值，$\mathbf{v}$ 为特征向量。PCA 降维的核心就是对协方差矩阵进行特征值分解。

3.5 奇异值分解 (SVD)

$\mathbf{A} = \mathbf{U} \boldsymbol{\Sigma} \mathbf{V}^T$，其中 $\mathbf{U}$ 和 $\mathbf{V}$ 是正交矩阵，$\boldsymbol{\Sigma}$ 是对角矩阵。SVD 广泛应用于推荐系统（协同过滤）和降维。

import numpy as np # 矩阵乘法 A = np.array([[1, 2], [3, 4]]) B = np.array([[5, 6], [7, 8]]) C = A @ B # 或 np.dot(A, B) # SVD分解 U, S, Vt = np.linalg.svd(A) print("奇异值:", S)

四、数学基础（下）：概率论、统计与微积分

4.1 概率基础

  • 条件概率：$P(A|B) = \frac{P(A \cap B)}{P(B)}$
  • 贝叶斯定理：$P(A|B) = \frac{P(B|A)P(A)}{P(B)}$，这是朴素贝叶斯分类器和很多概率模型的基础。
  • 独立事件：$P(A \cap B) = P(A)P(B)$

4.2 常见概率分布

分布概率密度/质量函数机器学习中的使用
高斯分布$f(x) = \frac{1}{\sigma\sqrt{2\pi}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$线性回归误差假设、高斯朴素贝叶斯
伯努利分布$P(X=1)=p, P(X=0)=1-p$二分类问题建模
多项分布$P(X_1=x_1,...,X_k=x_k)=\frac{n!}{x_1!...x_k!}p_1^{x_1}...p_k^{x_k}$文本分类、多分类

4.3 最大似然估计 (MLE)

给定数据 $\mathcal{D} = \{x_1, x_2, \ldots, x_n\}$，参数 $\theta$ 的最大似然估计为：

$$\hat{\theta}_{MLE} = \arg\max_\theta \prod_{i=1}^n P(x_i|\theta)$$

实际中常对似然函数取对数（对数似然），将乘法变为加法：

$$\hat{\theta}_{MLE} = \arg\max_\theta \sum_{i=1}^n \log P(x_i|\theta)$$

4.4 微积分与优化

梯度：函数 $f(\mathbf{x})$ 的梯度 $\nabla f(\mathbf{x})$ 指向函数增长最快的方向。在机器学习中，我们沿梯度的反方向更新参数以最小化损失函数。

梯度下降更新公式：

$$\theta_{t+1} = \theta_t - \eta \nabla_\theta J(\theta_t)$$

其中 $\eta$ 是学习率，$J(\theta)$ 是损失函数。

链式法则：若 $z = f(y)$ 且 $y = g(x)$，则 $\frac{dz}{dx} = \frac{dz}{dy} \cdot \frac{dy}{dx}$。这是反向传播算法的数学基础。

五、监督学习：线性回归

5.1 模型假设

线性回归假设目标值 $y$ 是输入特征 $\mathbf{x}$ 的线性函数：

$$h_\theta(\mathbf{x}) = \theta_0 + \theta_1 x_1 + \theta_2 x_2 + \cdots + \theta_n x_n = \boldsymbol{\theta}^T \mathbf{x}$$

其中 $\theta_0$ 是偏置项（截距），$\theta_1, \ldots, \theta_n$ 是各特征的权重。

5.2 损失函数：均方误差 (MSE)

$$J(\boldsymbol{\theta}) = \frac{1}{2m} \sum_{i=1}^{m} (h_\theta(\mathbf{x}^{(i)}) - y^{(i)})^2$$

系数 $\frac{1}{2}$ 是为了求导方便，消去平方项的系数2。

5.3 求解方法

正规方程 (Normal Equation)

$$\boldsymbol{\theta} = (\mathbf{X}^T\mathbf{X})^{-1} \mathbf{X}^T \mathbf{y}$$

优点：一步到位，无需迭代。缺点：计算 $(\mathbf{X}^T\mathbf{X})^{-1}$ 的时间复杂度为 $O(n^3)$，不适合大量特征。

梯度下降 (Gradient Descent)

$$\theta_j := \theta_j - \alpha \frac{\partial}{\partial \theta_j} J(\boldsymbol{\theta})$$

其中偏导数：$\frac{\partial}{\partial \theta_j} J(\boldsymbol{\theta}) = \frac{1}{m} \sum_{i=1}^{m} (h_\theta(\mathbf{x}^{(i)}) - y^{(i)}) x_j^{(i)}$

5.4 正则化

  • 岭回归 (Ridge, $L_2$)：$J(\boldsymbol{\theta}) = MSE + \lambda \sum_{j=1}^{n} \theta_j^2$
  • Lasso ($L_1$)：$J(\boldsymbol{\theta}) = MSE + \lambda \sum_{j=1}^{n} |\theta_j|$
  • 弹性网络 (Elastic Net)：结合 $L_1$ 和 $L_2$：$J(\boldsymbol{\theta}) = MSE + \lambda_1 \sum |\theta_j| + \lambda_2 \sum \theta_j^2$
from sklearn.linear_model import LinearRegression, Ridge, Lasso from sklearn.model_selection import train_test_split X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2) # 普通线性回归 lr = LinearRegression() lr.fit(X_train, y_train) # 岭回归 ridge = Ridge(alpha=1.0) ridge.fit(X_train, y_train) print("R² 分数:", lr.score(X_test, y_test))

六、监督学习：逻辑回归

6.1 从回归到分类

逻辑回归通过 Sigmoid 函数将线性输出映射到 $[0, 1]$ 区间，表示属于正类的概率：

$$g(z) = \frac{1}{1 + e^{-z}}$$ $$h_\theta(\mathbf{x}) = g(\boldsymbol{\theta}^T \mathbf{x}) = \frac{1}{1 + e^{-\boldsymbol{\theta}^T \mathbf{x}}}$$

6.2 决策边界

当 $h_\theta(\mathbf{x}) \geq 0.5$（即 $\boldsymbol{\theta}^T \mathbf{x} \geq 0$）时预测为正类；否则预测为负类。

6.3 损失函数：交叉熵

$$J(\boldsymbol{\theta}) = -\frac{1}{m} \sum_{i=1}^{m} \left[ y^{(i)} \log(h_\theta(\mathbf{x}^{(i)})) + (1-y^{(i)}) \log(1 - h_\theta(\mathbf{x}^{(i)})) \right]$$

为什么不用 MSE？对于逻辑回归，MSE 会导致非凸优化问题，容易陷入局部最优。交叉熵损失是凸函数，保证梯度下降能找到全局最优。

6.4 多分类扩展：Softmax 回归

对于 $K$ 类分类问题，Softmax 函数为每个类别计算概率：

$$P(y=k|\mathbf{x}) = \frac{e^{\boldsymbol{\theta}_k^T \mathbf{x}}}{\sum_{j=1}^{K} e^{\boldsymbol{\theta}_j^T \mathbf{x}}}$$
from sklearn.linear_model import LogisticRegression # 二分类 clf = LogisticRegression(penalty='l2', C=1.0, solver='lbfgs') clf.fit(X_train, y_train) # 多分类 clf_multi = LogisticRegression(multi_class='multinomial', solver='lbfgs') clf_multi.fit(X_train, y_train) print("准确率:", clf.score(X_test, y_test))

七、监督学习：决策树与随机森林

7.1 决策树原理

决策树通过递归地选择最优特征进行分裂，构建树形结构。每次分裂追求"纯度提升"最大化。

7.2 特征选择指标

信息增益 (ID3)

熵：$H(D) = -\sum_{k=1}^{K} p_k \log_2 p_k$

信息增益：$g(D, A) = H(D) - H(D|A) = H(D) - \sum_{v} \frac{|D^v|}{|D|} H(D^v)$

信息增益比 (C4.5)

$g_R(D, A) = \frac{g(D, A)}{H_A(D)}$，解决信息增益偏向取值较多特征的问题。

基尼不纯度 (CART)

$$Gini(D) = 1 - \sum_{k=1}^{K} p_k^2$$

基尼系数越小，数据集纯度越高。

7.3 剪枝策略

  • 预剪枝：在分裂前评估，若分裂不能带来泛化性能提升则停止
  • 后剪枝：先生成完整树，再自底向上剪去不重要的分支

7.4 随机森林

随机森林是 Bagging + 决策树的组合，通过双重随机性（样本随机采样 + 特征随机选择）来降低方差。

对于分类任务，随机森林通常取 $m = \sqrt{p}$ 个特征（$p$ 是总特征数），对于回归任务取 $m = p/3$。

from sklearn.tree import DecisionTreeClassifier from sklearn.ensemble import RandomForestClassifier # 单棵决策树 dt = DecisionTreeClassifier(max_depth=5, criterion='gini') # 随机森林 rf = RandomForestClassifier(n_estimators=100, max_depth=10, random_state=42) rf.fit(X_train, y_train) # 特征重要性 importances = rf.feature_importances_

八、监督学习：支持向量机 (SVM)

8.1 最大间隔原理

SVM 的目标是找到一个超平面 $\mathbf{w}^T\mathbf{x} + b = 0$，使得不同类别的样本到超平面的最小距离（间隔）最大化。

对于线性可分数据，超平面方程为：

$$\mathbf{w}^T \mathbf{x} + b = 0$$

间隔为 $\frac{2}{\|\mathbf{w}\|}$，最大化间隔等价于最小化 $\frac{1}{2}\|\mathbf{w}\|^2$。

8.2 软间隔与松弛变量

现实数据往往线性不可分，引入松弛变量 $\xi_i \geq 0$ 允许部分误分类：

$$\min_{\mathbf{w}, b, \boldsymbol{\xi}} \frac{1}{2}\|\mathbf{w}\|^2 + C \sum_{i=1}^{m} \xi_i$$

$C$ 是惩罚参数：$C$ 越大，对误分类惩罚越重，越容易过拟合；$C$ 越小，间隔越大，容错性越强。

8.3 核技巧

通过核函数隐式地将数据映射到高维特征空间，在高维空间中找到线性超平面：

$$K(\mathbf{x}_i, \mathbf{x}_j) = \phi(\mathbf{x}_i)^T \phi(\mathbf{x}_j)$$
核函数表达式适用场景
线性核$K(\mathbf{x}_i, \mathbf{x}_j) = \mathbf{x}_i^T \mathbf{x}_j$线性可分数据
多项式核$K(\mathbf{x}_i, \mathbf{x}_j) = (\mathbf{x}_i^T \mathbf{x}_j + c)^d$有一定非线性特征
高斯核 (RBF)$K(\mathbf{x}_i, \mathbf{x}_j) = \exp(-\gamma \|\mathbf{x}_i - \mathbf{x}_j\|^2)$通用、最常用
Sigmoid 核$K(\mathbf{x}_i, \mathbf{x}_j) = \tanh(\alpha \mathbf{x}_i^T \mathbf{x}_j + c)$类似神经网络
from sklearn.svm import SVC # RBF核SVM svm_rbf = SVC(kernel='rbf', C=1.0, gamma='scale', probability=True) svm_rbf.fit(X_train, y_train) # 获取支持向量 print("支持向量数量:", len(svm_rbf.support_vectors_))
调参建议：先使用线性核确定 baseline，再尝试 RBF 核。$\gamma$ 越大，决策边界越复杂，越容易过拟合。

九、监督学习：KNN 与朴素贝叶斯

9.1 K近邻算法 (KNN)

KNN 是典型的"惰性学习"算法——没有显式训练过程，预测时才计算样本与所有训练数据的距离。

距离度量

  • 欧氏距离：$d(\mathbf{x}, \mathbf{y}) = \sqrt{\sum_{i=1}^{n} (x_i - y_i)^2}$
  • 曼哈顿距离：$d(\mathbf{x}, \mathbf{y}) = \sum_{i=1}^{n} |x_i - y_i|$
  • 闵可夫斯基距离：$d(\mathbf{x}, \mathbf{y}) = (\sum_{i=1}^{n} |x_i - y_i|^p)^{1/p}$
  • 余弦相似度：$\cos(\mathbf{x}, \mathbf{y}) = \frac{\mathbf{x}^T \mathbf{y}}{\|\mathbf{x}\| \|\mathbf{y}\|}$

K值选择

$K$ 太小：模型复杂，容易过拟合（对噪声敏感）；$K$ 太大：模型简单，容易欠拟合（忽略局部特征）。通常通过交叉验证选择最优 $K$。

9.2 朴素贝叶斯

基于贝叶斯定理和"特征条件独立"假设（这是"朴素"的来源）：

$$P(y|x_1, x_2, \ldots, x_n) \propto P(y) \prod_{i=1}^{n} P(x_i|y)$$

常见变体

  • 高斯朴素贝叶斯：假设 $P(x_i|y) \sim \mathcal{N}(\mu_{y,i}, \sigma_{y,i}^2)$，适用于连续特征
  • 多项式朴素贝叶斯：适用于离散计数特征（如文本词频）
  • 伯努利朴素贝叶斯：适用于二值特征（如词是否出现）
注意：朴素贝叶斯的"特征独立"假设在现实中几乎不成立，但它依然在很多任务（尤其是文本分类）中表现良好，因为分类决策关注的是 $P(y|\mathbf{x})$ 的相对大小而非绝对值。
from sklearn.neighbors import KNeighborsClassifier from sklearn.naive_bayes import GaussianNB, MultinomialNB knn = KNeighborsClassifier(n_neighbors=5, weights='distance') knn.fit(X_train, y_train) gnb = GaussianNB() gnb.fit(X_train, y_train) print("KNN准确率:", knn.score(X_test, y_test)) print("Naive Bayes准确率:", gnb.score(X_test, y_test))

十、无监督学习：聚类与降维

10.1 K-Means 聚类

K-Means 是最经典的聚类算法，目标是最小化簇内平方和：

$$\min_{\mathbf{C}, \boldsymbol{\mu}} \sum_{k=1}^{K} \sum_{\mathbf{x}_i \in C_k} \|\mathbf{x}_i - \boldsymbol{\mu}_k\|^2$$

算法步骤：①随机初始化 $K$ 个中心点 → ②将每个样本分配给最近的中心点 → ③重新计算每个簇的中心点（均值）→ ④重复②③直到收敛。

K值选择

  • 肘部法则：绘制 $K$ 与 SSE（簇内误差平方和）的关系图，找到"拐点"
  • 轮廓系数：$s(i) = \frac{b(i) - a(i)}{\max(a(i), b(i))}$，取值范围 $[-1,1]$，越接近1越好

10.2 层次聚类

不需要预先指定簇数，生成树状图 (Dendrogram)。分为凝聚式（自底向上）和分裂式（自顶向下）。

10.3 DBSCAN

基于密度的聚类算法，能够发现任意形状的簇，并自动识别噪声点。核心参数：$\varepsilon$（邻域半径）和 $minPts$（最小点数）。

10.4 主成分分析 (PCA)

PCA 是线性降维方法，目标是找到数据方差最大的方向（主成分）：

步骤：①数据中心化 → ②计算协方差矩阵 $\mathbf{S} = \frac{1}{n-1}\mathbf{X}^T\mathbf{X}$ → ③对 $\mathbf{S}$ 做特征值分解 → ④取前 $k$ 大特征值对应的特征向量构成投影矩阵。

保留的方差比例：$\frac{\sum_{i=1}^{k} \lambda_i}{\sum_{i=1}^{n} \lambda_i}$

10.5 t-SNE

非线性降维技术，特别适合高维数据的可视化。保持数据点之间的局部相似性，将高维数据嵌入到2D或3D空间。注意：t-SNE 主要用于可视化，不适合作为特征提取步骤。

from sklearn.cluster import KMeans from sklearn.decomposition import PCA kmeans = KMeans(n_clusters=3, random_state=42, n_init=10) clusters = kmeans.fit_predict(X) # PCA降维到2维用于可视化 pca = PCA(n_components=2) X_pca = pca.fit_transform(X) print("保留方差比例:", pca.explained_variance_ratio_.sum())

十一、集成学习

11.1 为什么需要集成

单个模型可能存在偏差或方差问题。通过组合多个"弱学习器"，集成方法可以显著提升泛化性能。集成学习分为两大类：并行化方法（Bagging）和序列化方法（Boosting）。

11.2 Bagging (Bootstrap Aggregating)

核心思想：并行训练 $T$ 个基学习器，每个学习器使用自助采样（有放回）得到的不同训练集。最终通过投票（分类）或平均（回归）得到结果。

代表算法：随机森林——在 Bagging 基础上增加了特征随机选择。

11.3 Boosting

核心思想：串行训练基学习器，每个新学习器重点关注前一个学习器错误分类的样本。

AdaBoost

通过调整样本权重实现：错误分类的样本权重增加，正确分类的权重减少。

分类器权重：$\alpha_t = \frac{1}{2} \ln\left(\frac{1 - \epsilon_t}{\epsilon_t}\right)$

梯度提升 (GBDT)

每个新学习器拟合前一个学习器的残差（负梯度方向）：

$$F_m(\mathbf{x}) = F_{m-1}(\mathbf{x}) + \nu \cdot h_m(\mathbf{x})$$

其中 $\nu$ 是学习率（ shrinkage），$h_m$ 是拟合残差的决策树。

XGBoost

在 GBDT 基础上做了大量工程优化：

  • 目标函数加入正则项：$Obj = \sum_i L(y_i, \hat{y}_i) + \sum_k \Omega(f_k)$
  • 二阶泰勒展开，利用二阶导数信息
  • 支持列采样、缺失值自动处理
  • 支持并行化特征分裂

LightGBM

基于直方图的决策树算法，使用单边梯度采样 (GOSS) 和互斥特征捆绑 (EFB)，训练速度更快，内存占用更低。

11.4 Stacking

训练多个不同的基模型（第一层），用它们的预测输出作为元模型（第二层）的输入特征，学习最优的组合方式。

from sklearn.ensemble import RandomForestClassifier, GradientBoostingClassifier import xgboost as xgb rf = RandomForestClassifier(n_estimators=100) gbdt = GradientBoostingClassifier(n_estimators=100, learning_rate=0.1) xgb_model = xgb.XGBClassifier(n_estimators=100, learning_rate=0.1) for model in [rf, gbdt, xgb_model]: model.fit(X_train, y_train) print(f"{model.__class__.__name__}: {model.score(X_test, y_test):.4f}")

十二、特征工程实战

12.1 数据清洗

  • 缺失值处理：删除（缺失比例高）、填充（均值/中位数/众数/模型预测）、标记
  • 异常值处理：IQR法（$Q1-1.5 \times IQR$ 到 $Q3+1.5 \times IQR$）、Z-score法（$|Z| > 3$）、截断或转换
  • 重复值：直接删除或保留一条

12.2 特征缩放

  • 标准化 (Standardization)：$x' = \frac{x - \mu}{\sigma}$，使特征均值为0、方差为1
  • 归一化 (Min-Max Scaling)：$x' = \frac{x - x_{min}}{x_{max} - x_{min}}$，映射到 $[0,1]$
  • 鲁棒缩放 (Robust Scaling)：$x' = \frac{x - median}{IQR}$，对异常值鲁棒

12.3 特征编码

  • 独热编码 (One-Hot)：$K$ 个类别生成 $K$ 维向量，适用于类别数少的无序变量
  • 标签编码 (Label)：将类别映射为整数 $0,1,\ldots,K-1$，适用于有序变量或树模型
  • 目标编码 (Target Encoding)：用目标变量的均值替换类别值，需注意防止数据泄露

12.4 特征构造

  • 多项式特征：从 $x_1, x_2$ 构造 $x_1^2, x_2^2, x_1x_2$ 等
  • 交互特征：组合两个特征的乘积或比值
  • 分箱：将连续变量离散化
  • 时间特征：从日期提取年、月、日、星期、季度等

12.5 特征选择

方法类型示例

包裹法 (Wrapper)模型性能递归特征消除 (RFE)、前向/后向搜索

十三、模型评估与超参数调优

13.1 分类指标

定义混淆矩阵：

预测正类预测负类
实际正类TP (True Positive)FN (False Negative)
实际负类FP (False Positive)TN (True Negative)
  • 准确率：$Accuracy = \frac{TP + TN}{TP + TN + FP + FN}$
  • 精确率：$Precision = \frac{TP}{TP + FP}$（预测为正的有多少是真的）
  • 召回率：$Recall = \frac{TP}{TP + FN}$（真正的正类有多少被找出来）
  • F1 分数：$F_1 = 2 \cdot \frac{Precision \cdot Recall}{Precision + Recall}$
  • ROC-AUC：ROC 曲线下的面积，衡量模型整体区分正负类的能力，$AUC \in [0.5, 1]$

13.2 回归指标

  • MSE：$MSE = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$
  • RMSE：$RMSE = \sqrt{MSE}$
  • MAE：$MAE = \frac{1}{n}\sum_{i=1}^{n}|y_i - \hat{y}_i|$
  • $R^2$：$R^2 = 1 - \frac{\sum(y_i - \hat{y}_i)^2}{\sum(y_i - \bar{y})^2}$，越接近1越好

13.3 交叉验证

$K$ 折交叉验证：将数据均匀分成 $K$ 份，每次用 $K-1$ 份训练、1份验证，重复 $K$ 次取平均。常用 $K=5$ 或 $K=10$。分层 $K$ 折确保每折类别比例一致。

13.4 偏差-方差权衡

  • 高偏差（欠拟合）：模型太简单，训练误差和验证误差都高 → 增加模型复杂度、添加特征
  • 高方差（过拟合）：训练误差低但验证误差高 → 增加数据、正则化、简化模型

13.5 超参数调优

  • 网格搜索 (Grid Search)：穷举所有参数组合，适合小参数空间
  • 随机搜索 (Random Search)：随机采样参数空间，效率更高
  • 贝叶斯优化：使用代理模型（如高斯过程）建模参数与性能的关系，智能选择下一组参数
from sklearn.model_selection import GridSearchCV, cross_val_score param_grid = { 'n_estimators': [50, 100, 200], 'max_depth': [5, 10, None], 'min_samples_split': [2, 5, 10] } rf = RandomForestClassifier(random_state=42) grid = GridSearchCV(rf, param_grid, cv=5, scoring='f1', n_jobs=-1) grid.fit(X_train, y_train) print("最佳参数:", grid.best_params_) print("最佳分数:", grid.best_score_)

十四、神经网络基础

14.1 感知机

感知机是最简单的神经网络单元：$f(\mathbf{x}) = sign(\mathbf{w}^T\mathbf{x} + b)$。但单层感知机只能解决线性可分问题（无法处理 XOR）。

14.2 多层感知机 (MLP)

通过堆叠多个全连接层，引入非线性激活函数，MLP 能够逼近任意函数（万能逼近定理）。

一层隐藏层的 MLP：$\mathbf{h} = g(\mathbf{W}_1\mathbf{x} + \mathbf{b}_1)$，$\hat{\mathbf{y}} = \mathbf{W}_2\mathbf{h} + \mathbf{b}_2$

14.3 激活函数

函数公式优点缺点
Sigmoid$\sigma(x) = \frac{1}{1+e^{-x}}$平滑、输出(0,1)梯度消失、非零中心
Tanh$\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$零中心、输出(-1,1)梯度消失
ReLU$f(x) = \max(0, x)$计算快、缓解梯度消失神经元"死亡"(负输入梯度0)
Leaky ReLU$f(x) = \max(\alpha x, x)$解决死亡ReLU引入了超参数 $\alpha$

14.4 前向传播与反向传播

前向传播：输入数据逐层通过网络的权重矩阵和非线性激活函数，最终输出预测值。

反向传播：利用链式法则计算损失函数对每个参数的梯度：

$$\frac{\partial L}{\partial \mathbf{W}^{(l)}} = \frac{\partial L}{\partial \mathbf{a}^{(l)}} \cdot \frac{\partial \mathbf{a}^{(l)}}{\partial \mathbf{z}^{(l)}} \cdot \frac{\partial \mathbf{z}^{(l)}}{\partial \mathbf{W}^{(l)}}$$

然后使用梯度下降或其变体更新参数：$\mathbf{W} \leftarrow \mathbf{W} - \eta \nabla_\mathbf{W} L$

14.5 损失函数

  • 回归：MSE Loss = $\frac{1}{n}\sum(y_i - \hat{y}_i)^2$；MAE Loss = $\frac{1}{n}\sum|y_i - \hat{y}_i|$
  • 二分类：BCE Loss = $-\frac{1}{n}\sum[y_i\log\hat{y}_i + (1-y_i)\log(1-\hat{y}_i)]$
  • 多分类：CrossEntropy Loss = $-\frac{1}{n}\sum\sum y_{ik}\log\hat{y}_{ik}$

14.6 优化器

  • SGD：$\theta_{t+1}=\theta_t - \eta g_t$，简单但收敛慢
  • SGD + Momentum：$v_{t+1}=\beta v_t + (1-\beta)g_t$，$\theta_{t+1}=\theta_t - \eta v_{t+1}$，加速收敛
  • Adam：结合 Momentum 和 RMSProp，自适应学习率，是目前最常用的优化器

14.7 正则化技术

  • $L_1/L_2$ 权重衰减：在损失中加入 $\lambda\|\mathbf{W}\|^2$
  • Dropout：训练时随机丢弃 $p$ 比例的神经元，防止过拟合
  • Batch Normalization：对每层输入进行标准化，加速训练，有一定正则化效果

十五、卷积神经网络 (CNN)

15.1 为什么需要 CNN

全连接网络处理图像时面临两大问题：①参数量爆炸（$224\times224\times3$ 的图像第一层就有1500万+参数）；②忽略了图像的空间局部结构。CNN 通过局部连接和权值共享解决了这两个问题。

15.2 核心组件

卷积层

卷积核在输入上滑动，执行逐元素乘加操作。卷积操作：

$$(f * g)(t) = \sum_{\tau} f(\tau) g(t-\tau)$$

输出尺寸：$O = \frac{W - K + 2P}{S} + 1$，其中 $W$ 是输入尺寸，$K$ 是卷积核大小，$P$ 是填充，$S$ 是步长。

池化层

降采样操作，减小特征图尺寸，增加平移不变性。最大池化是最常用的池化方式。

15.3 经典架构演进

模型年份核心创新Top-5错误率
LeNet-51998首个CNN应用（手写数字识别）—

VGG-162014全部使用3×3小卷积核，深度增加7.3%

15.4 残差网络 (ResNet) 详解

核心思想：引入跳跃连接（Skip Connection），让网络学习残差映射：

$$\mathbf{y} = \mathcal{F}(\mathbf{x}, \{W_i\}) + \mathbf{x}$$

当 $\mathcal{F}=0$ 时，$\mathbf{y}=\mathbf{x}$，恒等映射。这种设计解决了深层网络的退化问题（层数增加但性能反而下降），使训练数百层的网络成为可能。

import torch.nn as nn # 简单的CNN class SimpleCNN(nn.Module): def __init__(self, num_classes=10): super().__init__() self.conv1 = nn.Conv2d(3, 32, 3, padding=1) self.conv2 = nn.Conv2d(32, 64, 3, padding=1) self.pool = nn.MaxPool2d(2, 2) self.fc1 = nn.Linear(64 * 8 * 8, 512) self.fc2 = nn.Linear(512, num_classes) self.relu = nn.ReLU() self.dropout = nn.Dropout(0.5) def forward(self, x): x = self.pool(self.relu(self.conv1(x))) x = self.pool(self.relu(self.conv2(x))) x = x.view(x.size(0), -1) x = self.relu(self.fc1(x)) x = self.dropout(x) return self.fc2(x)

十六、循环神经网络 (RNN / LSTM / GRU)

16.1 RNN 基础

RNN 通过隐藏状态 $h_t$ 将前一时间步的信息传递到当前步：

$$h_t = \tanh(\mathbf{W}_{ih} x_t + \mathbf{W}_{hh} h_{t-1} + b_h)$$ $$y_t = \mathbf{W}_{hy} h_t + b_y$$

16.2 RNN 的问题

梯度消失/爆炸：由于反向传播通过时间 (BPTT) 涉及连乘操作，当序列较长时，早期时间步的梯度要么趋于零（梯度消失），要么趋于无穷大（梯度爆炸）。这导致 RNN 无法捕捉长距离依赖关系。

16.3 LSTM (长短期记忆)

LSTM 引入细胞状态 $C_t$ 和三个门控机制，有效解决梯度消失问题：

  • 遗忘门：$f_t = \sigma(W_f \cdot [h_{t-1}, x_t] + b_f)$ —— 决定丢弃哪些旧信息
  • 输入门：$i_t = \sigma(W_i \cdot [h_{t-1}, x_t] + b_i)$，$\tilde{C}_t = \tanh(W_C \cdot [h_{t-1}, x_t] + b_C)$ —— 决定存储哪些新信息
  • 细胞状态更新：$C_t = f_t * C_{t-1} + i_t * \tilde{C}_t$
  • 输出门：$o_t = \sigma(W_o \cdot [h_{t-1}, x_t] + b_o)$，$h_t = o_t * \tanh(C_t)$

16.4 GRU (门控循环单元)

GRU 是 LSTM 的简化版，合并了遗忘门和输入门为单一的更新门，参数更少，训练更快：

  • 更新门：$z_t = \sigma(W_z \cdot [h_{t-1}, x_t])$
  • 重置门：$r_t = \sigma(W_r \cdot [h_{t-1}, x_t])$
  • 候选隐藏状态：$\tilde{h}_t = \tanh(W \cdot [r_t * h_{t-1}, x_t])$
  • 最终隐藏状态：$h_t = (1 - z_t) * h_{t-1} + z_t * \tilde{h}_t$

16.5 应用场景

  • 文本分类/情感分析：LSTM/GRU + 全连接层
  • 机器翻译：Seq2Seq（编码器-解码器架构）
  • 时间序列预测：股票价格、天气预测
  • 命名实体识别 (NER)：BiLSTM + CRF
import torch.nn as nn class SentimentLSTM(nn.Module): def __init__(self, vocab_size, embed_dim, hidden_dim, num_classes): super().__init__() self.embedding = nn.Embedding(vocab_size, embed_dim) self.lstm = nn.LSTM(embed_dim, hidden_dim, num_layers=2, batch_first=True, bidirectional=True) self.fc = nn.Linear(hidden_dim * 2, num_classes) self.dropout = nn.Dropout(0.5) def forward(self, x): x = self.embedding(x) out, (h_n, c_n) = self.lstm(x) # 取最后时间步的双向拼接 out = torch.cat((h_n[-2,:,:], h_n[-1,:,:]), dim=1) out = self.dropout(out) return self.fc(out)

十七、Transformer 与注意力机制

17.1 自注意力 (Self-Attention)

Transformer 的核心创新是完全抛弃了 RNN 的序列结构，仅依赖注意力机制。自注意力计算每个位置与其他所有位置的相关性：

$$\text{Attention}(\mathbf{Q}, \mathbf{K}, \mathbf{V}) = \text{softmax}\left(\frac{\mathbf{Q}\mathbf{K}^T}{\sqrt{d_k}}\right) \mathbf{V}$$

其中 $\mathbf{Q} = \mathbf{X}\mathbf{W}^Q$，$\mathbf{K} = \mathbf{X}\mathbf{W}^K$，$\mathbf{V} = \mathbf{X}\mathbf{W}^V$。除以 $\sqrt{d_k}$ 是为了防止点积过大导致 softmax 梯度消失。

17.2 多头注意力 (Multi-Head Attention)

并行计算多个注意力"头"，每个头关注不同的表示子空间：

$$\text{MultiHead}(\mathbf{Q}, \mathbf{K}, \mathbf{V}) = \text{Concat}(\text{head}_1, \ldots, \text{head}_h) \mathbf{W}^O$$ $$\text{head}_i = \text{Attention}(\mathbf{Q}\mathbf{W}_i^Q, \mathbf{K}\mathbf{W}_i^K, \mathbf{V}\mathbf{W}_i^V)$$

17.3 Transformer 架构

编码器 (Encoder)：由 $N$ 个相同的层堆叠，每层包含：多头自注意力 → 残差连接+层归一化 → 前馈网络 → 残差连接+层归一化。BERT 使用编码器部分。

解码器 (Decoder)：除了与编码器相同的两个子层，还增加了交叉注意力层（对编码器输出做注意力）。使用掩码自注意力确保预测当前位置时只能看到之前的位置。GPT 系列使用解码器部分。

17.4 位置编码

由于 Transformer 没有 RNN 的序列处理能力，需要显式注入位置信息：

$$PE_{(pos, 2i)} = \sin\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$ $$PE_{(pos, 2i+1)} = \cos\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$

十八、生成模型：GAN 与 VAE

18.1 生成对抗网络 (GAN)

GAN 由生成器 $G$ 和判别器 $D$ 组成，两者进行极小极大博弈：

$$\min_G \max_D V(D, G) = \mathbb{E}_{x \sim p_{data}}[\log D(x)] + \mathbb{E}_{z \sim p_z}[\log(1 - D(G(z)))]$$
  • 生成器：从随机噪声 $z$ 生成逼真的假样本，目标是骗过判别器
  • 判别器：区分真实样本和假样本

18.2 GAN 的训练难点

  • 模式坍塌 (Mode Collapse)：生成器只产生少数几种样本
  • 训练不稳定：生成器和判别器的平衡难以维持
  • 梯度消失：判别器太强时，生成器梯度接近零

18.3 变分自编码器 (VAE)

VAE 学习数据的潜在分布 $p(z|x)$，通过最大化证据下界 (ELBO) 来训练：

$$\mathcal{L} = \mathbb{E}_{q(z|x)}[\log p(x|z)] - D_{KL}(q(z|x) \| p(z))$$

第一项是重建损失，第二项是 KL 散度正则项，迫使潜在分布接近标准正态分布。

注意：GAN 的理论优雅但工程实现困难，是深度学习中最难训练的模型之一。初学者建议先从 VAE 入手理解生成模型的基本框架。

十九、深度学习框架：PyTorch 与 TensorFlow

19.1 PyTorch

由 Meta AI 开发，以动态计算图为核心特色。代码风格贴近 Python 原生语法，调试方便，是学术界的首选框架。

核心概念

  • Tensor：多维数组，与 NumPy 类似但支持 GPU 加速和自动微分
  • autograd：自动微分引擎，记录所有操作并自动计算梯度
